# WeatherWise
    It is an weather app which will provide the real-time weather of the given location.

    It provides the information like maximum_temperature, minimum_tempreature, wind speed, sunrise_time, sunset_time.

    It also give an insight of the maximum_temperature and minimum_tempreature of the coming one week.

## How to use
    The Ui is very comprehensive, as there is an search bar which will take any city name and you will bw able to see the weather insights.
